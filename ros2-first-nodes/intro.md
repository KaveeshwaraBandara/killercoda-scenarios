# Your first ROS 2 nodes — for real

This is a **real Linux machine** running in your browser. In the next few minutes you will:

- start a **talker** node that publishes messages to a topic called `/chatter`
- start a **listener** node that hears every one of them
- eavesdrop on the topic yourself with the ROS 2 command-line tools

While you were reading this, the environment began installing ROS 2 Jazzy in the background — it'll be ready by the time you need it.

This scenario is part of the free course **[Learn ROS 2 from Zero](https://kaveeshwarabandara.github.io/ros2-course/)** — head there for the full learning path.

Click **START** when you're ready.
